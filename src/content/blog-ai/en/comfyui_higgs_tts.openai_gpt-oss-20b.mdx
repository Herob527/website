# Introduction

In this article you’ll learn how to use **ComfyUI** together with a text‑to‑speech (TTS) system called **HiggsTTS 3**.

ComfyUI itself does not include a TTS node, but the [TTS Audio Suite](https://github.com/diodiogod/TTS-Audio-Suite) adds many nodes that connect different TTS engines. After testing several options, the **Higgs Audio v3 Engine** proved to be the best for generating Polish‑style speech, so we’ll focus on it.

## Installation

If the ComfyUI Manager does not work, install the TTS package manually:

```bash
# Go to the custom_nodes folder of your ComfyUI installation
cd /path/to/comfy-ui/custom_nodes

# Clone the TTS Audio Suite repository
git clone https://github.com/diodiogod/TTS-Audio-Suite

# Return to the root directory
cd ..

# Install required Python packages
uv pip install -r custom_nodes/tts_audio_suite/requirements.txt

# Start ComfyUI so the new nodes appear
uv run main.py
```

You should see a progress bar while the TTS engines load.

## Prompting HiggsTTS 3

HiggsTTS 3 lets you change the voice’s style and emotions by inserting special tokens in your text. These tokens appear between `|<` and `>|`.  
For example: `<|emotion:joy|>Hello world!`

A full list of available tokens is found in the [HiggsTTS3 repository on Huggingface](https://huggingface.co/bosonai/higgs-tts-3-4b).  

Because HiggsTTS can produce slightly different results for the same text and seed, try generating multiple times with different seeds to get the best sounding output.  
You can also force an English voice to sound Polish; however, the quality may vary.

#### Style
| Token | Description |
|------|-------------|
| `<\|style:singing\|>` | Singing |
| `<\|style:shouting\|>` | Shouted voice |
| `<\|style:whispering\|>` | Whispered voice |

#### Sound Effects
| Token | Description | Suggested onomatopoeia |
|------|-------------|------------------------|
| `<\|sfx:cough\|>` | Cough | Ahem |
| `<\|sfx:laughter\|>` | Laughter | Haha / Hehe |
| `<\|sfx:crying\|>` | Crying | Boohoo / Sob |
| `<\|sfx:screaming\|>` | Screaming | Ahh / Aaah |
| `<\|sfx:burping\|>` | Burping | Burp |
| `<\|sfx:humming\|>` | Humming | Hmm / Mmm |
| `<\|sfx:sigh\|>` | Sigh | Uh / Ahh |
| `<\|sfx:sniff\|>` | Sniff | Sff |
| `<\|sfx:sneeze\|>` | Sneeze | Achoo |

#### Prosody
| Token | Effect |
|------|--------|
| `<\|prosody:speed_very_slow\|>` | ≈0.65× speed |
| `<\|prosody:speed_slow\|>` | ≈0.85× speed |
| `<\|prosody:speed_fast\|>` | ≈1.2× speed |
| `<\|prosody:speed_very_fast\|>` | ≈1.4× speed |
| `<\|prosody:pitch_low\|>` | ≈−3 semitones |
| `<\|prosody:pitch_high\|>` | ≈+2.5 semitones |
| `<\|prosody:pause\|>` | ≈400–700 ms pause |
| `<\|prosody:long_pause\|>` | ≈700–1500 ms pause |
| `<\|prosody:expressive_high\|>` | More expressive delivery |
| `<\|prosody:expressive_low\|>` | Flatter delivery |
